### _NOTE_
_cara install/gunakan less_<br>

>```
> npm install -g less
>```
>masuk ke file css
>cd css
>gunakan perintah
> ```
> lessc styles.less styles.css
> ```
<b>lessc</b> : _perintah untuk compile file .less_ <br>
<b>styles.less</b> : _ambil file .less_ <br>
<b>styles.css</b> : _hasil compile file .less menjadi .css_ <br>
